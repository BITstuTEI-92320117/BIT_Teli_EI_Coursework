# 一、整体功能
本 MATLAB 代码实现数字基带传输中的码型变换与性能分析，支持二进制随机序列生成、AMI/HDB3 归零编码、时域波形绘制及频域功率谱估计，可完成固定 / 可变 “0” 概率下的码型性能对比实验。

# 二、函数说明
- `gen_seq(N, p)`：生成长度为 N、“0” 出现概率为 p 的二进制随机序列。
- `AMI_enc(seq, fs)`：对二进制序列 seq 进行归零 AMI 编码（“1” 交替 ±1，“0” 为 0），采样率 fs，输出编码序列及归零波形。
- `HDB3_enc(seq, fs)`：对二进制序列 seq 进行归零 HDB3 编码（限制连 “0”≤3，插入破坏脉冲），采样率 fs，输出编码序列及归零波形。
- `NRZ(code, ls, fs)`：非归零编码，将码型序列转换为占空比 100% 的非归零波形（ls 为序列长度）。
- `RZ(code, ls, fs)`：归零编码，将码型序列转换为占空比 50% 的归零波形（采样率 fs）。
- `draw(seq, code, level, code_type, N_draw, fs)`：绘制前 N_draw 位信源序列、编码序列及归零波形，code_type 指定码型（AMI/HDB3）。

# 三、主脚本说明
- 实验切换：通过 exp 值选择实验类型（exp=1 为实验1，exp≠1 为实验1）。
- 实验1（exp=1）：固定“0”概率 p=0.7，完成 AMI/HDB3 编码、前 20 位波形绘制及原始序列/AMI/HDB3 码的功率谱对比。
- 实验2（exp=2）：测试“0”概率为 0.1/0.3/0.5/0.7/0.9 时，三种码型的功率谱变化规律。

# 四、使用说明
- 运行主脚本，通过修改 exp 值切换实验类型。
- 可调整参数：N（序列长度）、fs（采样率）、nfft（FFT 长度）等。
- 仿真结果（波形图、功率谱对比图）自动保存为本地图片文件。
