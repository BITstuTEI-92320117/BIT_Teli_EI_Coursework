# 北理工特立电信资料
## 项目说明
* 本仓库为北京理工大学特立书院电子信息工程专业学习资料/大作业/往年题汇总整理。
* 本仓库所有资料由特立电信23级几位不愿意透露姓名的热心学长共同提供。
* 除少数资料为wzh学长之前免费提供，绝大部分是几位学长共同完成与收集。
* 本仓库资料仅针对特立电信提供，睿信电信或其他专业可能仅部分有用。由于能力确实有限，部分资料可能存在不合理之处，请注意甄别。
* 本仓库资料全部无偿提供，后续随课程进行仍会持续更新~<br>

**注意：新一届特立电信的培养方案已发生变化，部分资料可能与实际不符。欢迎学弟学妹在讨论区留言指出，也欢迎Fork原仓库分享最新的课程资料，负责人会及时对本仓库进行更新。**

## 大二下
**1. 数值计算方法**<br>
&emsp;&emsp;没啥好说的，这门课主要就平时作业与考勤，提供资料包含课本和前几年往年题。<br>
**2. 数字电路与系统**<br>
&emsp;&emsp;提供的资料包含课本、老师划的重点、往年题和实验报告。往年题部分由wzh学长提供，实验部分Verilog代码源文件之前未保存，但可借鉴报告中提供的代码。<br>
**3. 数据结构与算法设计(C++描述)**<br>
&emsp;&emsp;提供的资料包含乐学所有C++编程题源代码和大作业的源代码、实验报告和汇报PPT。由于当时选择银行账户、贪吃蛇和压缩软件这三题，因此只提供对应部分资料。<br>
**4. 控制理论基础**<br>
&emsp;&emsp;提供的资料包含实验报告、可运行MATLAB源代码和往年题，所有实验代码可直接运行。<br>
**5. 信号处理理论与技术I**<br>
&emsp;&emsp;提供的资料包含仿真实验、翻转课堂和往年题。往年题应该是睿信信号与系统课程的，与特立这边的应该会有所出入。仿真实验包含实验报告和对应的MATLAB代码，将对应部分的代码取消注释可直接运行。翻转课堂的要求是选择一个信号处理相关的课题进行研究并课堂汇报展示，不限制具体内容。由于当时时间仓促并且未学过通信原理，所选课题相对简略且存在一些不合理之处，仅供参考。<br>
**6. 通信与网络基础理论及应用I**<br>
&emsp;&emsp;提供的资料包含平时作业答案、雨课堂小测答案和研究型学习PPT和源代码，唯一遗憾之处是未搜集到往年题。去年研究型学习前两题是必做，当时选择算术编码和LDPC编码这两题，因此只提供对应部分资料。第三题深度学习为选做，代码是利用deepseek辅助编写完成，运行需配置深度学习所需环境并安装对应的库。<br>
**7. 电磁理论、计算、应用I**<br>
&emsp;&emsp;由于任课老师取消了大作业，考核方式只有平时作业和期中期末小测。但听说以前都是有大作业的，不知道以后有没有。提供一份wzh学长之前的实验报告，以及自己搜集的睿信电磁场往年题。


## 大三上
**1. 计算机原理与应用**<br>
&emsp;&emsp;提供的资料包含汇编上机考试自己写的代码、两份平时作业、硬件实验报告和往年题。往年题部分是自动化或其他专业的，请注意甄别。<br>
**2. 信号处理理论与技术II**<br>
&emsp;&emsp;提供的资料包含作业答案、仿真实验、翻转课堂和往年题。往年题应该是睿信数字信号处理的，难度比特立这边大一些。仿真实验包含实验报告和对应的MATLAB代码，所有代码可直接运行，翻转课堂要求与信号I类似。<br>
**3. 通信与网络基础理论及应用II**<br>
&emsp;&emsp;提供的资料包含平时作业答案、往年题和实验报告以及源代码，往年题大部分是睿信通信原理的题目，考试范围与特立的存在一定不同。<br>
**4. 电磁理论、计算、应用II**<br>
&emsp;&emsp;提供的资料包含矩量法、预警机虚拟仿真实验和HFSS仿真实验三个大作业的实验报告、源代码和仿真设计，以及来自于wzh学长的一份平时作业答案。矩量法、预警机虚拟仿真实验部分借鉴wzh学长的思路，HFSS仿真实验在助教的帮助下独立完成。<br>
**5. 信号与信息处理课程设计**<br>
&emsp;&emsp;提供的资料包含实验讲义、实验报告、源代码和实验数据，源代码为MATLAB APP设计文件，所有GUI软件系统可直接运行。<br>
