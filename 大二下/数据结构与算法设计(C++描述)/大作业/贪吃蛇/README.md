# 贪吃蛇游戏
## 一、项目简介
基于 C++ 开发的经典贪吃蛇游戏，支持**无限模式**（无障碍物、可调节速度与分值）和**闯关模式**（含障碍物、关卡升级），依赖 Windows API 实现控制台交互，通过颜色区分蛇身、食物、障碍物等元素，操作简单易上手。

## 二、运行方法
1. 环境要求：Windows 7 及以上系统 + 支持 C++11 及以上标准的编译器（推荐 Visual Studio 2022）；
2. 编译步骤：将 `贪吃蛇.cpp` 导入编译器，直接编译生成可执行文件（无需额外依赖库）；
3. 启动游戏：运行生成的 exe 文件，在欢迎界面按「1」选择无限模式、按「2」选择闯关模式，进入游戏后按方向键开始控制。

## 三、文件说明
| 文件名       | 用途                                   |
|--------------|----------------------------------------|
| 贪吃蛇.cpp   | 唯一源代码文件，整合蛇节点/障碍物类、游戏逻辑函数及主程序，实现完整玩法 |

## 四、核心函数用途
| 函数名                  | 用途                                   |
|-------------------------|----------------------------------------|
| SetColor(int color)     | 设置控制台文本颜色（区分蛇头、食物等元素） |
| Pos(int x, int y)       | 定位光标位置，实现精准绘图（避免闪屏）   |
| creatMap()              | 绘制游戏边界墙（白色方块）               |
| initsnake()             | 初始化4节蛇身并绘制（蛇头红、蛇身白）   |
| createfood()            | 随机生成食物（绿色，不与蛇身重叠）       |
| createObstacles()       | 闯关模式专属，生成蓝色障碍物（不冲突）   |
| moveSnake(int newX, int newY) | 处理蛇移动/增长（吃食物变长，否则平移） |
| snakemove()             | 蛇移动主逻辑，整合撞墙、自噬检测         |
| biteself()/cantcrosswall() | 检测蛇自噬、撞墙（触发则游戏结束）     |
| hitObstacle()           | 闯关模式检测蛇撞障碍物（触发则游戏结束） |
| checkLevelUp()          | 闯关模式得分达标后升级（障碍物增多、蛇加速） |
| pause()                 | 空格键暂停/继续游戏，暂停时ESC可退出     |
| gamecircle()            | 游戏主循环，处理键盘输入与状态更新       |
| welcometogame()/endgame() | 欢迎界面（模式选择）/ 结束界面（显示得分与原因） |
| main()                  | 程序入口，整合初始化、游戏启动流程       |

## 五、操作指南
| 按键         | 功能描述                          | 适用场景               |
|--------------|-----------------------------------|------------------------|
| ↑/↓/←/→      | 控制蛇移动方向（不可反向，如向右时不能直接向左） | 所有模式（运行中）     |
| 空格键       | 暂停/继续游戏                     | 所有模式（运行中）     |
| ESC 键       | 主动退出游戏（暂停时也可使用）    | 所有模式               |
| F1 键        | 加速移动 + 增加食物分值（每次+2，上限20） | 仅无限模式             |
| F2 键        | 减速移动 + 减少食物分值（每次-2，下限1） | 仅无限模式             |
| F3 键        | 重置速度（200ms）与食物分值（10分） | 仅无限模式             |
| 1/2 键       | 选择无限模式/闯关模式              | 欢迎界面               |

## 六、注意事项
1. 仅支持 Windows 系统，Linux/macOS 需修改控制台交互相关代码（依赖 `windows.h`）；
2. 蛇移动时不可反向（如当前向右，按左键无效），避免触发自噬判定；
3. 闯关模式共5关，每关需累计当前关卡×100分升级，障碍物最多12个，蛇速最低50ms（最快）；
4. 游戏结束条件：撞墙、蛇自噬、闯关模式撞障碍物、主动按ESC退出；

5. 编译时若提示头文件缺失，需确保编译器支持 Windows API 相关库（默认支持）。
