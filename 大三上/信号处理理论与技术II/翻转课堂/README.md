# 任意点数FFT算法仿真
## 项目简介
本代码基于MATLAB实现，旨在探究任意点数DFT的高效求解算法，对比基2-FFT、基p-FFT、混合基FFT、雷德-混合基FFT与MATLAB内置fft函数、直接DFT计算的性能差异，验证不同算法的正确性与运算效率。

## 功能说明
1. 生成有限长单边指数序列作为仿真输入；
2. 绘制序列的DTFT（离散时间傅里叶变换）频谱图；
3. 实现5种DFT/FFT计算方式：
   - MATLAB内置fft函数（基准对比）；
   - 直接DFT计算（基于定义的平凡实现）；
   - 基p-FFT算法（支持指定质数p，需序列长度为p的幂次）；
   - 混合基FFT算法（支持任意复合数长度，基于质因数分解）；
   - 雷德-混合基FFT算法（优化质数/含大质因子长度的计算）；
4. 输出各算法的运算时间、频谱图及均方误差（与内置fft结果对比）。

## 依赖环境
- 运行环境：MATLAB（兼容R2018b及以上版本）；
- 无额外第三方工具依赖。

## 使用方法
1. 下载所有代码文件并置于同一MATLAB工作目录；
2. 打开主脚本（无单独文件名，直接运行包含`clc,clear,close all;`的主代码）；
3. （可选）修改仿真参数：
   - `L`：序列长度（支持质数如3001、复合数如3000、p的幂次如2187=3⁷、3125=5⁵）；
   - `p`：基p-FFT的基数（需为质数，且仅当`Radix_p_flag=1`时生效）；
   - 算法开关（`Radix_p_flag`/`Radix_mix_flag`/`Radix_rader_mix_flag`）：1=启用，0=禁用；
4. 直接运行主脚本，自动输出运算时间、频谱图（保存为PNG文件）及误差信息。

## 文件说明
| 文件名                | 功能描述                                  |
|-----------------------|-------------------------------------------|
| 主脚本（无单独文件名）| 程序入口，负责参数设置、序列生成、算法调用及结果输出 |
| `dft.m`               | 直接DFT计算（基于定义的双重循环实现）      |
| `gen_seq.m`           | 生成单边指数序列 `x[n] = aⁿu[n]`          |
| `plot_dtft.m`         | 绘制DTFT频谱图                            |
| `plot_fft.m`          | 绘制各算法的FFT频谱图（子图布局）          |
| `RaderFFT.m`          | 雷德算法实现（处理质数长度序列）          |
| `Radix_2_FFT.m`/`Radix_2_IFFT.m` | 基2-FFT及逆变换（雷德算法依赖） |
| `Radix_p_FFT.m`       | 基p-FFT算法实现                            |
| `Radix_mix_fft.m`     | 混合基FFT算法实现（基于质因数分解）        |
| `Radix_rader_mix_fft.m` | 雷德-混合基FFT算法（融合混合基与雷德算法） |
| 辅助函数（`modExp.m`等） | 快速模幂、原根查找等工具函数（雷德算法依赖） |

## 注意事项
1. 基p-FFT算法启用时（`Radix_p_flag=1`），需确保序列长度`L`为`p`的整数幂（如p=3时，L=3、9、27...）；
2. 雷德-混合基FFT算法自动识别质数长度序列，小质数（≤32）直接用定义计算，大质数调用雷德算法；
3. 频谱图自动保存为`dtft.png`和`FFT频谱图.png`，保存在当前工作目录；
4. 运算时间受序列长度影响显著，质数长度（如3001）的直接DFT计算耗时较长，建议按需禁用无关算法开关。
