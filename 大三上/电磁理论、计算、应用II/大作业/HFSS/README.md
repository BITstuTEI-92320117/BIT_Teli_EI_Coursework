# HFSS仿真
1. 基于HFSS电磁仿真软件进行的微带相控阵天线论文复现，仿真模型的搭建除少数部分由课程助教帮助完成，绝大部分为本人亲自搭建。
2. HFSS作业为从四篇推荐论文中选择一篇利用HFSS进行仿真复现。对于微带天线相控阵论文复现，阵列单元的复现是必做内容，阵列扫描是选做内容。
2. 提供的内容包括论文复现仿真实验报告，搭建的HFSS设计模型和相关python代码，生成的微带相控阵天线波束扫描示意图以及主要参考教材。
3. 模型的搭建方法在报告中已详细阐述，可根据报告中的步骤进行建模。
4. 仿真计算结果由于占用空间较大在此不再提供，需要先分析计算才能得到报告中的仿真结果。
5. 报告中的所有图像均由MATLAB利用导出的仿真计算结果绘制，绘图代码运行需修改相应的文件路径。
